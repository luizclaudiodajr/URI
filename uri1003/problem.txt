Read two integer values in the case for variables A and B. Next, calculate the sum between them and assign the variable SUM. Then write this variable.

Entrance

The input file contains two integer values.

Output

Print SOMA variable in all capital letters, with a blank space before and after the equality followed by the amount corresponding to the sum of A and B. Like all problems, be sure to print the finish line after the result, otherwise you will receive "Presentation Error".
